![HIBC_BLOCKF barcode example](images/hibc_blockf.png)

Code example:
```python
import pyzint

symbol = pyzint.Barcode.HIBC_BLOCKF('123456789')

with open('HIBC_BLOCKF.bmp', "wb") as bmp:
    bmp.write(symbol.render_bmp())
```
